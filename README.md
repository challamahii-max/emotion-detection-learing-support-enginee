# 🎓 Emotion Detection Learning Support Engine

An AI-powered learning support system that detects students' emotions from text inputs and provides personalized learning assistance, motivational guidance, and emotional support to enhance the overall learning experience.

---

## 📖 Project Overview

The **Emotion Detection Learning Support Engine** is designed to identify a learner's emotional state and recommend appropriate learning resources and support. By leveraging Natural Language Processing (NLP) and Machine Learning techniques, the system helps create a more engaging and personalized educational environment.

This project follows a complete Software Development Life Cycle (SDLC), including brainstorming, requirement analysis, design, development, testing, documentation, and project demonstration.

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/Emotion-Detection-Learning-Support-Engine.git
```

### Navigate to the Project

```bash
cd Emotion-Detection-Learning-Support-Engine/Learning\ Support\ Engine
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

The application will start on your local machine.
